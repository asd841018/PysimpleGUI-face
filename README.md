# PysimpleGUI-Face-recognition

Face racognition with Arcface model and Face Anti spoofing.

## Setup
### Prerequisites
* Python 3.9.12
### Installation
```bash
pip install -r requirements.txt
```

### Data preparation

```
├── faces
    └── RGB_Images
        ├── name_1.jpg
		├── name_2.jpg
		└── ...
```
